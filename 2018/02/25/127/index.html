Description

给定一个长度为N的数组A[]，求有多少对i, j, k（1<=i<j<k<=N）满足A[k]-A[j]=A[j]-A[i]。

Input

第一行一个整数N（N<=10^5）。
接下来一行N个数A[i]（A[i]<=30000）。

Output

一行一个整数。

Sample Input

Sample Output

分块FFT
先分块， 暴力求出有三个在同一块，和两个在同一块的答案
三个都不在一块的FFT
