    你来到了一个庙前，庙牌上有一个仅包含小写字母的字符串 s。
    传说打开庙门的密码是这个字符串的一个子串 t，并且 t 既是 s 的前缀又是 s 的后缀并且还在 s 的中间位置出现过一次。

    如果存在这样的串，请你输出这个串，如有多个满足条件的串，输出最长的那一个。
    如果不存在这样的串，输出”Just a legend”(去掉引号)。

    输入格式：

    仅一行，字符串 s。

    输出格式：

    如题所述

    样例输入

    fixprefixsuffix

    样例输出：

    fix

    数据范围：

    对于 60%的数据， s 的长度<=100
    对于 100%的数据， s 的长度<=100000

    题解

    hash 淼之
