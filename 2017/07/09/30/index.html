    输入


    修正一下

    l = (l_0 + x - 1) mod n + 1, r = (r_0 + x - 1) mod n + 1

    输出

    样例输入

    6 3
    1 2 3 2 1 2
    1 5
    3 6
    1 5

    样例输出

    1
    2
    1

    修正下：
    n <= 40000, m <= 50000

    题解

    f(i,j)表示第 i 块到第 j 块的众数。
    为每一个数开vector<> 存他的位置
    在求得时候二分就可以了
    详见《区间众数解题报告 - 陈立杰》
